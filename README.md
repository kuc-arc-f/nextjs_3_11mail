### Setup

* npm install

* next.config.js , smtp setting
```
SMTP_HOST : "smtp.test.com",
SMTP_PORT : 465,
SMTP_SECURE : true,
SMTP_AUTH_USER : "user123",
SMTP_AUTH_PASS : "123",
SEND_MAIL_ADDRESS : "hoge@test.com"
```
***
### start server
* Start :

```
yarn dev
```
